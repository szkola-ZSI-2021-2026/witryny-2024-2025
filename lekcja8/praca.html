MIN-WIDTH

OPIS
Określa minimalną szerokość emelentu.

ZASTOSOWANIE
Gdy szerokość elementu jest mniejsza niż wartość ustawiona w min-width, przeglądarka nie zmniejszy szerokości elementu poniżej tej wartości.

PRZYKŁAD
div {
min-width: 200px;
}

MAX-WIDTH

OPIS
Określa minimalną szerokość elementu.

ZASTOSOWANIE
Gdy szerokość elementu jest większa niż wartość ustawiona w max-width, przeglądarka nie rozciągnie elementu ponad tę wartość.

PRZYKŁAD
div {
max-width: 500px;
}

MIN-HEIGHT

OPIS
Określa minimalną wysokość elementu.

ZASTOSOWANIE
Jeśli zawartość elementu wymaga mniejszej wysokości niż ta ustawiona w min-height, to przeglądarka ustawi wysokość elementu na tę wartość.

PRZYKŁAD
div {
min-height: 300px;
}

MAX-HEIGHT

OPIS
Określa maksymalną wysokość elementu.

ZASTOSOWANIE
Gdy wysokość elementu przekroczy wartość ustawioną w max-height, element nie zostanie rozciągnięty powyżej tej wartości.

PRZYKŁAD
div {
max-height: 600px;
}

MIN-WIDTH

OPIS
Określa, co ma się stać, gdy zawartość elementu przekroczy jego rozmiary (szerokość lub wysokość). Może przyjmować różne wartości:

  • visible (domyślnie) – zawartość wychodzi poza element, ale nie jest ukryta ani przewijana.
  • hidden – zawartość wychodzi poza element, ale jest ukryta (nie będzie widoczna).
  • scroll – zawartość jest przewijalna zarówno w poziomie, jak i w pionie, niezależnie od tego, czy występuje nadmiar zawartości.
  • auto – pojawiają się paski przewijania tylko wtedy, gdy zawartość wykracza poza rozmiary elementu.

PRZYKŁAD
div {
overflow: auto;
}

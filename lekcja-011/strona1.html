Magiczny świat HTML, CSS i JavaScript

HTML: Fundamenty Magicznego Domu

HTML, najstarsza z trójki, była mądra i pełna energii. Jej zadaniem było budowanie szkieletów stron internetowych. Niczym architekt, projektowała i konstruowała budynki - definiowała, gdzie mają być umieszczone nagłówki, akapity, obrazy i przyciski. Dzięki niej wszystkie strony miały solidne podstawy.

CSS: Artysta i Dekorator

CSS, młodsza i pełna kreatywności, była artystką, która kochała kolory, czcionki i uklady. Za pomocą magicznych pędzli, zwanych kaskadowymi arkuszami stylów, CSS nadawała każdemu elementowi estetyczny wygląd. Na przykład, oto jak CSS nadaje kolor tła:
body {
background-color: lightblue;
}
CSS również porządkowała zawartość strony, nadając styl nagłówkom i akapitom:
h1 {
color: darkblue;
font-family: Arial, sans-serif;
text-align: center;
}
P{
font-size: 16px;
color: darkgray;
line-height: 1.5;
margin: 20px;
}

JavaScript: Carodziejka Animacji

Na końcu pojawiła się JavaScript, najmłodsza z przyjaciółek, ale również najdynamiczniejsza. Była czarodziejką, która dodawała interaktywność i animacje. Razem z CSS stworzyła niesamowity efekt przycisków:
.button {
background-color: darkblue; color: white;
padding: 10px 20px;
border: none;
cursor: pointer;
transition: background-color 0.3s; }
.button:hover { background-color: white;
color:navy;
box-shadow: 2px 2px 4px rgba(0, 0, 0,0.2);
border-left:1px solid lightgrey;
border-top: 1px solid lightgrey;
}
